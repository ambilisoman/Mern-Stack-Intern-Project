React
- React is a JavaScript library used to build user interfaces.
- It helps us create reusable UI components and update data without reloading the page.

SPA (Single Page Application)
- A SPA loads one HTML page, and content updates dynamically using JavaScript.
- Faster
- Smooth user experience
- No full page reloads
- Examples: Gmail, Instagram, Facebook

Node.js
- Runtime environment to run JavaScript outside the browser
- Required to run React tools

npm (Node Package Manager)
- Comes with Node.js
- Used to install libraries and dependencies
